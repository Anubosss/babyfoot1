BABY-FOOT connecté

Le Baby-foot connecté permet de faciliter le comptage de point lors d'une partie. Il est capable de reconnaitre les simples points , les demi , trimi et les gammelles.
Il est équipé d'ecrans pour afficher les points des 2 équipes. Il a aussi des capteurs de mouvements pour detecter la balle dans les cages. Et pour finir , il est équipé de boutons pour définir le type de points marqués.
Son prix est de 1000 euros. C'est un baby-foot intérieur, idéal pour toute la famile !
